DictoSaurusMixin

DictoSaurusMixin class Null safety

A mixin class that implements the Dictionary, Thesaurus and AutoCorrect interface methods as well as DictoSaurus.expandTerm.

Implementations that mix in DictoSaurusMixin must override dictionary, autoCorrect and Thesaurus.
