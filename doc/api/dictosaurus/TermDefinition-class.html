TermDefinition

TermDefinition class Null safety

A definition for a term, used in a TermProperties object with immutable properties:

  • term is a term or word;
  • partOfSpeech is the PartOfSpeech category of the term;
  • definition is the definition for term when used as partOfSpeech;
  • synonyms is an unordered collection of unique terms that are synonyms of term when used as partOfSpeech;
  • antonyms is an unordered collection of unique terms that are antonyms of term when used as partOfSpeech;
  • inflections is an unordered collection of unique terms that are inflections of term when used as partOfSpeech; and
  • phrases is an unordered collection of unique example phrases that include term when used as partOfSpeech.
