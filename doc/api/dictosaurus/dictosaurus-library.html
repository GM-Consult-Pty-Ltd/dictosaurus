Extensions on String that provide dictionary and thesaurus functions.

Classes

The AutoCorrect class exposes the suggestionsFor function that returns a set of unique alternative spellings for a term.

A mixin class that implements AutoCorrect.suggestionsFor.
Thesaurus
The Thesaurus class exposes the synonymsOf function that asynchronously returns the synonyms of a term from a SynonymsIndex.
Vocabulary
The Vocabulary class exposes the definitionOf function, looking up the meaning of a term in a VocabularyIndex;
