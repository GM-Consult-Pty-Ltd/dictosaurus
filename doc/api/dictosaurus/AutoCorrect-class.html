AutoCorrect

AutoCorrect class Null safety

The AutoCorrect interface exposes the suggestionsFor and startsWith methods.

  • suggestionsFor returns a linked set of unique alternative spellings for a term and its tokenized versions.
  • startsWith returns an ordered list of unique terms that start with a sequence of characters from a k-grams index.
