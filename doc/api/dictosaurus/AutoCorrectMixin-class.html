AutoCorrectMixin

AutoCorrectMixin class Null safety

A mixin class that implements the AutoCorrect.suggestionsFor and AutoCorrect.startsWith methods.

Classes that mix in AutoCorrectMixin must override:

  • k, the length of the k-grams in the Map<String, Set<String>>; and
  • kGramIndexLoader, an asynchronous callback function that returns a Map<String, Set<String>> for a collection of k-grams.
